# Wrist & Biases

![Wrist & Biases](resources/banner.png)

Monitor your [Weights & Biases](https://wandb.ai) ML training runs from your Pebble smartwatch.

## Features

- View all your W&B runs organized by state (running, crashed, finished)
- Browse metrics with real-time values and historical graphs
- Scrub through metric history to see past values
- Smooth animations and intuitive navigation
